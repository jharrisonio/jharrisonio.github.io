Generating target accounts from job ads

October 9, 2022 · 2 min read
James Harrison
Solution Architect

Introduction​

The FL0 platform is almost ready to be introduced to the world and I need to generate a list of companies that might be interested in trying it out. This presents a few challenges:

  1. What is the criteria for finding suitable companies?
  2. Where is there a large enough database of companies that is easily accessible?
  3. Why would they be interested in what I have to say?

The criteria was fairly easy to identify. Any companies we contact must be using NodeJS, as our platform is based on this technology. Ideally they would be a smaller company that is able to implement change and try new software easily without a lot of red tape. Any familiarity with similar platforms like Vercel would be a bonus, as would a current growth or restructuring phase. In summary:

  • NodeJS
  • Fewer than 100 employees
  • Familiarity with hosting/CI/CD automation tools
  • Growing or restructuring

The tech stack​

I chose to use Puppeteer - "a Node library which provides a high-level API to control Chrome or Chromium over the DevTools Protocol.". To extract elements from the returned HTML, I used Cheerio. I bundled these into a friendly CLI using Inquirer.js. This is my first time building a proper CLI tool with Javascript, so I followed this tutorial.
